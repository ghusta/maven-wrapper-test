# Test : maven-wrapper

## Installation de maven-wrapper

> mvn -N io.takari:maven:wrapper

## Avantage 

- Plus besoin d'installer une version Maven en local. 

## Utilisation

Au lieu de la commande `mvn ...` on utilisera `./mvnw ...` (unix) ou `./mvnw.cmd ...` (windows).
